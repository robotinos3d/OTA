#Actualizaciones por OTA
Al cambiar el número de version en el archivo `version.txt` a uno mayor, todos los archivos dentro de la carpeta `/app` serán actualizados a los microcontroladores.
Antes de pushear un cambio, se recomienda:
<ol>
  <li>Cambiar la version en el archivo `version.txt`</li>
  <li>Ejecutar el script `setup.bash`</li>
</ol>
